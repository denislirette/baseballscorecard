Release History

v0.3.0 15 March 2026

Branding

  • Renamed from BaseballScorebook to BaseballScorecard.org across all pages, titles, and meta tags
  • Recalibrated version numbers to proper semver (v0.x instead of v1/v2)
  • Removed MLB team logos (trademark compliance) — team names and abbreviations used instead

Design System

  • Adopted design tokens from denislirette.com: link colors, visited states, typography scale, heading weights
  • Fluid body text sizing with clamp() for responsive scaling
  • All font sizes converted from px/rem to em for consistent relative scaling
  • Offwhite background (#faf9f6) replaces pure white throughout
  • Inverted header bar: dark background in light mode, light in dark mode
  • Global link styles: blue unvisited, purple visited, thicker underline on hover

Navigation

  • Two-row header: site title bar + nav bar with hamburger menu on mobile
  • Dark/light mode toggle in header bar with sun/moon icons
  • Renamed Games to Scorecards in nav
  • External link icon on links that open in new tabs
  • Combined Legend and Advanced Stats into a single Reference page

Reference Page

  • Left sidebar subnav with Wikipedia-style section hierarchy and scroll spy
  • Batting and pitching formulas as stacked definition lists (name, formula, description)
  • LaTeX-rendered formulas via MathJax

Game Picker

  • Custom accessible date picker with ARIA grid pattern and full keyboard navigation
  • Game cards now use CSS grid (left-to-right flow) instead of CSS columns
  • Fixed font size on game cards to prevent team name wrapping
  • Game times now include timezone (e.g., "7:05 PM EDT")

Footer

  • Subtle border using color-mix transparency
  • Version number in grey
  • Streamlined links: Contact, Accessibility

Infrastructure

  • GitHub issue template converted to YAML form with structured fields
  • Blank issues disabled — users must use the bug report form
  • Local-only style panel (gitignored) for live font-size tweaking

v0.2.0 14 March 2026

Scorecard

  • Play cell redesign: grid-based layout with pitches on the right, diamond and notation on the left, consistent 10px inner padding
  • Pitch display simplified: removed call code letter, now shows just pitch type + velocity (e.g., "FF 95")
  • Two-column pitch overflow when an at-bat exceeds 10 pitches
  • ABS challenge tracking: purple W/L indicators on challenged pitches, purple dots in strike zone diagram
  • Pitcher sub redesign: square blocks instead of dashed lines, with S/P/K stats centered on the boundary line
  • DP notation always splits across two lines in both full scorecard and thumbnails
  • Bat-around summary rows: sub-column vertical lines no longer extend into merged summary area
  • Pickoff distinction: PO (pickoff) now separate from CS (caught stealing) in diamond annotations
  • All 80+ MLB event types handled: complete coverage including pickoffs, caught stealing, balks, interference

Game Page

  • Team comparison table: W-L, GB, Home, Road, division records, interleague, L10, and streak
  • Game info tables: game details, weather, and umpires as consistent tables
  • WP/LP/SV shown as plain text next to the linescore
  • Bench table now matches bullpen layout with full stats (AVG, OBP, SLG, OPS, wOBA, HR, RBI, SB, PA)
  • Pitcher repertoire: starting pitcher info shows usage % and average velocity per pitch type
  • wOBA calculated client-side from MLB API raw stats using 2024 FanGraphs linear weights

Thumbnails

  • 20% larger cells, two-line DP notation, active cell highlighting
  • Abbreviations only, divider line removed, thinner grey card borders

Site

  • Advanced Stats reference page with LaTeX-rendered formulas
  • Legend page: converted from popup overlay to standalone page
  • Standings page: converted from popup overlay to standalone page
  • Global navigation: dynamic nav module injected on every page
  • Sun/moon SVG icons for theme toggle
  • Repo cleanup: removed 21 unnecessary files, moved docs to docs/ folder

v0.1.0 7 March 2026

Initial Release

  • Full SVG scorecard rendering
  • GUMBO feed parsing for lineups, play notation, and runner tracking
  • Pitch sequence display with strike zone plot
  • Substitution indicators (pitcher, pinch hitter, pinch runner)
  • Auto-refresh for live games
  • Light/dark theme support
  • Game picker with date navigation and scorecard thumbnails
  • Fixture-driven development mode
