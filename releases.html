Release History

v2.0.0

14 March 2026

  • Play cell redesign: New grid-based layout with pitches on the right, diamond and notation on the left, consistent 10px inner padding, and proper alignment system for all elements
  • Pitch display simplified: Removed call code letter, now shows just pitch type + velocity (e.g., "FF 95")
  • Two-column pitch overflow: When an at-bat exceeds 10 pitches, pitches split into two columns automatically
  • ABS challenge tracking: Purple W/L indicators on challenged pitches, purple dots in strike zone diagram
  • Team comparison table: W-L, GB, Home, Road, division records, interleague, L10, and streak at the top of every game page
  • Game info tables: Game details, weather, and umpires now render as consistent tables using the shared pitcher-stats-table style
  • Decisions display: WP/LP/SV shown as plain text next to the linescore, not in a separate table
  • Pitcher sub redesign: Square blocks instead of dashed lines, with S/P/K stats centered on the boundary line
  • Bench table: Now matches bullpen layout with full stats (AVG, OBP, SLG, OPS, wOBA, HR, RBI, SB, PA)
  • wOBA calculation: Computed client-side from MLB API raw stats using 2024 FanGraphs linear weights
  • Advanced Stats page: New reference page with LaTeX-rendered formulas for all batting and pitching stats
  • Legend page: Converted from popup overlay to standalone page with all current abbreviations
  • Standings page: Converted from popup overlay to standalone page
  • Global navigation: Dynamic nav module (nav.js) injected on every page for consistent Legend, Standings, Stats, and theme toggle buttons
  • Theme toggle: Sun/moon SVG icons replace text labels
  • All 80+ MLB event types handled: Complete coverage of every eventType from the MLB Stats API including pickoffs, caught stealing, balks, interference, and more
  • Thumbnail improvements: 20% larger cells, two-line DP notation, active cell highlighting, abbreviations only, divider line removed, thinner grey card borders
  • Pickoff distinction: PO (pickoff) now separate from CS (caught stealing) in diamond annotations
  • Pitcher repertoire format: Starting pitcher info now shows usage % and average velocity per pitch type
  • Repo cleanup: Removed 21 unnecessary files (PROJECT_BRIEF, NOTES, unused Figma pipeline, stubs), moved docs to docs/ folder, updated .gitignore
  • DP notation: Always splits across two lines in both full scorecard and thumbnails
  • Bat-around summary rows: Sub-column vertical lines no longer extend into merged summary area
