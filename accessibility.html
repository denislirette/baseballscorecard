Accessibility

Accessibility Statement

BaseballScorebook.org is committed to making baseball scorecard data accessible to all users.

What we do

  • WCAG AA compliant color contrast in both light and dark themes
  • All interactive elements are keyboard accessible
  • SVG scorecards include aria-labels for screen readers
  • Date picker supports keyboard input (DD-MM-YYYY format)
  • Color is never the sole indicator of meaning; text labels accompany all color-coded elements
  • Responsive layout works across desktop, tablet, and mobile viewports

Known limitations

  • SVG scorecard cell-level data is visual and not fully navigable by screen readers. We are working on improving this.
  • The mini strike zone plot relies on color-coded dots. Pitch call codes in the text column provide the same information in text form.

Feedback

If you encounter accessibility barriers, please open an issue on GitHub. We take accessibility feedback seriously and will work to address it.
